# H5P Content Types - Complete Reference

## Overview
H5P offers 50+ content types. This reference covers the most useful ones for educational contexts.

## Interactive Content Types

### 1. Interactive Video
**Description:** Video with embedded questions, text popups, and navigation points

**Best for:**
- Flipped classroom lessons
- Video comprehension checks
- Tutorial walkthroughs
- Documentary analysis

**Key Features:**
- Add questions at specific timestamps
- Text/image overlays
- Bookmarks for navigation
- Adaptive branching based on answers
- Summary screen with results

**Example Use Cases:**
- Science experiments with comprehension questions
- Historical documentaries with context popups
- Language learning with vocabulary hints
- Math tutorials with pause-and-practice

**Pro Tips:**
- Keep videos under 10 minutes
- Add questions every 2-3 minutes
- Use bookmarks for key concepts
- Provide immediate feedback

---

### 2. Quiz (Question Set)
**Description:** Collection of various question types in sequence

**Question Types Included:**
- Multiple Choice
- True/False
- Fill in the Blanks
- Drag and Drop
- Drag the Words
- Mark the Words

**Best for:**
- End-of-unit assessments
- Self-check quizzes
- Homework assignments
- Formative assessments

**Key Features:**
- Mix multiple question types
- Randomize questions
- Set pass percentage
- Show solutions
- Retry option

**Pro Tips:**
- Start easy, increase difficulty
- Mix question types for engagement
- Provide explanations for wrong answers
- Allow multiple attempts for learning

---

### 3. Course Presentation
**Description:** Slide-based presentation with embedded interactive elements

**Embedded Elements:**
- Text, images, videos
- Questions
- Interactive videos
- Drag and drop
- Fill in blanks

**Best for:**
- Digital lessons
- Self-paced learning modules
- Presentations with checkpoints
- Guided tutorials

**Key Features:**
- Navigation controls
- Bookmarks
- Summary slide
- Print functionality
- Keywords for search

**Pro Tips:**
- One concept per slide
- Mix media types
- Add questions every 3-5 slides
- Use summary slide for review

---

### 4. Interactive Book
**Description:** Multi-page digital book with various H5P elements

**Best for:**
- Digital textbooks
- Comprehensive course modules
- Self-contained learning units
- Reference materials

**Key Features:**
- Table of contents
- Multiple pages/chapters
- Embed any H5P content
- Progress tracking
- Summary page

**Pro Tips:**
- Structure like real textbook
- Use chapters logically
- Include practice throughout
- Add summary at chapter ends

---

### 5. Timeline
**Description:** Interactive chronological presentation

**Best for:**
- Historical events
- Project timelines
- Process flows
- Biographical content

**Key Features:**
- Date-based organization
- Images and text
- Zoom and navigation
- Multimedia integration

**Pro Tips:**
- Use for sequences of 5+ events
- Include relevant images
- Add context to each point
- Consider alternative formats for simple sequences

---

### 6. Drag and Drop
**Description:** Interactive categorization and matching exercises

**Types:**
- Text to zones
- Images to zones
- Labels to images
- Sequential ordering

**Best for:**
- Vocabulary matching
- Categorization
- Labeling diagrams
- Sorting exercises

**Key Features:**
- Multiple drop zones
- Feedback per item
- Show correct answers
- Retry option

**Pro Tips:**
- Clear visual distinction between zones
- Not too many items (5-10 ideal)
- Provide immediate feedback
- Use images when possible

---

## Quick Reference: Content Type Selection

**Need assessment?** → Quiz / Question Set
**Have a video?** → Interactive Video
**Teaching a process?** → Course Presentation or Timeline
**Want practice activities?** → Drag and Drop, Fill in Blanks
**Creating course materials?** → Interactive Book
**Vocabulary practice?** → Dialog Cards, Memory Game
**Simple text organization?** → Accordion

## Combination Strategies

### Strategy 1: Complete Lesson
1. Interactive Video (introduction)
2. Course Presentation (detailed content)
3. Quiz (assessment)

### Strategy 2: Self-Paced Module
1. Interactive Book (theory)
2. Drag and Drop (practice)
3. Question Set (check understanding)

### Strategy 3: Flipped Classroom
1. Interactive Video (homework - watch at home)
2. In-class: Discussion + advanced activities
3. Quiz (after class - verify understanding)

## Mobile Considerations

**Works well on mobile:**
- Multiple Choice
- True/False
- Interactive Video
- Quiz
- Timeline

**Challenging on mobile:**
- Complex Drag and Drop (small screens)
- Detailed Interactive Books (scrolling)
- Course Presentations with tiny text

**Best Practice:** Test on actual mobile devices before publishing

## Accessibility Notes

- Always include alt text for images
- Provide transcripts for videos
- Use sufficient color contrast
- Test with screen readers
- Allow keyboard navigation
